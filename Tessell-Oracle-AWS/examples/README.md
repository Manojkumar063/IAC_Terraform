# Examples

This folder will contain example configurations for different use cases:

## Planned Examples

- `single-instance/` — Basic single instance setup (current implementation)
- `high-availability/` — HA configuration with multiple nodes
- `multi-database/` — Multiple databases in one service
- `production/` — Production-ready configuration with all security features
- `disaster-recovery/` — DR setup with cross-region replication

## Contributing

Add your example configurations here with:
1. Complete Terraform files
2. README explaining the use case
3. terraform.tfvars.example with required variables
